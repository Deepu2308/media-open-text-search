# Media finder
Open text media search

![demo](src/data/examples/demo-video.gif)


# Setup
Clone repo, cd into it and run 
```
make build
```

# Interface
Start a UI to search for media in repo
```
make interface
```

# Debug/Dev with Jupyter notebook
```
make run
```
